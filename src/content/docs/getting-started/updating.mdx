# Daemon

## Guided

Download the Azul repository once again, and run the install script for your OS (`install-windows.ps1`, `install-macos-linux.sh`) to automatically update to the latest version.

## Manually

To update the Azul Daemon manually, you can use `git` to get the latest changes from the Azul repository.

Open your terminal in the Azul repository folder and run:

```
git pull
```

# Plugin

## Automatically

Updating the Azul Plugin is as simple as clicking the "Update" button in Studio: `Plugins > Manage Plugins > Azul > Update`.

## Manually

If you installed the Plugin manually, you have to remove the old version from your plugins folder & [get the latest version](https://github.com/Ransomwave/azul/tree/main/plugin#method-2-manual-install-via-place-file).
