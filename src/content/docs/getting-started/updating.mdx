---
title: Updating Azul
description: Update Azul to the latest version.
---

## Update the CLI

### If installed with npm (most users)

Run:

```bash
npm install -g azul-sync@latest
```

Then verify:

```bash
azul --version
```

### If installed from source

1. Pull latest changes in your local clone.
2. Rebuild and reinstall globally.

```bash
git pull
npm install
npm run build
npm install -g .
```

## Update the Studio plugin

### Automatic update

In Studio, open **Plugins > Manage Plugins > Azul > Update**.

### Manual update

If you installed a local/manual plugin copy, remove the old version and install the latest plugin build from:

https://github.com/Ransomwave/azul/tree/main/plugin
