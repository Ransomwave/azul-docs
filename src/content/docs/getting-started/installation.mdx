Azul has 2 very important components:

- the Daemon, which runs on your machine and handles syncing
- the Plugin, which runs in Roblox Studio and communicates with the Daemon.

To use Azul, you need to install both components:

# Daemon

## Install Script (Recommended)

1. Clone the Azul repository or [download it as a ZIP](https://github.com/Ransomwave/azul/archive/refs/heads/main.zip) and extract it.
2. Inside the extracted folder, locate the `/scripts` Folder
3. Run the install script for your OS (`install-windows.ps1`, `install-macos-linux.sh`).
4. Check Azul is working by running `azul -h` in your terminal. You should see the help message printed out.
5. _(Optional) [Set up recommended tools](#recommended-tools) for the best experience._

## Manual Install

1. Clone the repository or download the ZIP and extract it.
2. Install Node.js from [nodejs.org](https://nodejs.org/) or by using your system's package manager:
   ```bash
   # Windows (using winget)
   winget install OpenJS.NodeJS.LTS
   # macOS (using Homebrew)
   brew install node
   # Linux (using apt)
   sudo apt install nodejs npm
   ```
3. Install dependencies by running
   ```
   npm install
   ```
4. Build the project
   ```
   npm run build
   ```
5. Install the project globally
   ```
   npm install -g .
   ```

# Recommended tools

While these tools aren't required to use Azul, they are highly recommended to get the best experience out of Azul:

### Luau-LSP

Use the [Luau Language Server extension](https://marketplace.visualstudio.com/items?itemName=JohnnyMorganz.luau-lsp) for [VSCode](https://code.visualstudio.com/). This extension provides Luau IntelliSense, error checking, and more.

It works by reading the `sourcemap.json` file generated by Azul to understand the structure of your project and provide accurate suggestions and error checking.

To get Luau-LSP working with Azul, open your `User Settings (JSON)` from the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`) and make sure to set up luau-lsp like this:

```json
  "luau-lsp.plugin.enabled": true,
  "luau-lsp.sourcemap.enabled": true,
  "luau-lsp.sourcemap.autogenerate": false,
  "luau-lsp.sourcemap.sourcemapFile": "sourcemap.json",
  "luau-lsp.sourcemap.includeNonScripts": true,
  "luau-lsp.sourcemap.useVSCodeWatcher": true,
```

Luau-LSP is also available for other IDEs like [IntelliJ](https://plugins.jetbrains.com/plugin/24957-luau) and [Neovim](https://github.com/lopi-py/luau-lsp.nvim), but setup may vary.

### Verde (VSCode Extension)

[Verde](https://marketplace.visualstudio.com/items?itemName=Dvitash.verde) is a VSCode extension that mimics the Roblox Studio Explorer and Properties windows. It works great alongside Azul to provide a seamless development experience.
